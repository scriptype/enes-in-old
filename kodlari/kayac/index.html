Kayaç

  • jQuery İle Çalışan Slider

  • İsteğe Bağlı Otomatik Geçiş Özelliği

  • Temiz, Basit, Sade

  • Düğmeler Dışarı Alınabilir

  • Youtube Videoları İle Uyumlu

Özellikler

  • Ücretsiz.
  • Tüm tarayıcılarda sorunsuz çalışır.
  • En uygun Kayaç'ı elde etmek için ayar parametreleriyle oynayın.
  • Panellere istenilen her türden kod parçası yerleştirilebilir.
  • Youtube videoları (iframe) için optimize edildi.
  • Otomatik geçiş özelliği kapatılabilir.
  • Düğmeler resim dışına alınabilir ya da kaldırılabilir.
  • Kenarlara önceki-sonraki panele geçiş düğmeleri eklenebilir.
  • var ayar = {
  • otoGec: true, // false: otomatik geçmesin
  • gosterSure: 5000, // 1000: 1 saniye, x000: x saniye
  • gecSure: 800, // geçiş efektinin süresi
  • genislik: 960,
  • yukseklik: 480,
  • sagSolDugme: true, // slider'ın kenarlarında sağa-sola ilerletme amaçlı oklar çıkıp çıkmaması.
  • panel: false, // true: butonları resmin dışına al. 20px + buton yüksekliği kadar fazladan yer kaplar.
  • dugme: true // false: düğmeleri kaldır. bu durumda video izlenmeye başlandığında video panelinden çıkış yolu kalmayacaktır.
  • }
